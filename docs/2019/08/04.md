1. 5种Js解构运算的一些用法

	[https://dmitripavlutin.com/5-interesting-uses-javascript-destructuring/](https://dmitripavlutin.com/5-interesting-uses-javascript-destructuring/)

	```js
	// 传统变量值交换
	let a = 1;
	let b = 2;
	let temp;

	temp = a;
	a = b;
	b = temp;

	a; // => 2
	b; // => 1

	// 解构
	let a = 1;
	let b = 2;

	[a, b] = [b, a];

	a; // => 2
	b; // => 1
	```
