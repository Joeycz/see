8. 猎奇：文言文編程語言

	[https://github.com/LingDong-/wenyan-lang](https://github.com/LingDong-/wenyan-lang)

	> 吾有一數。曰三。名之曰「甲」。
	>
	> 為是「甲」遍。
	>
	>	吾有一言。曰「「問天地好在。」」。書之。
	>
	> 云云。

	相当于
	
	```js
	var n = 3;
	for (var i = 0; i < n; i++) {
		console.log("問天地好在。");
	}
	```
